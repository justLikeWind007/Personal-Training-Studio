# 安全巡检清单 v1

巡检脚本：`./scripts/security_baseline_check.sh`

## 发布前

- [ ] 已为 Gateway/Biz/Ops 配置同一份 `INTERNAL_TOKEN_VALUE`
- [ ] 非生产环境先保持 `INTERNAL_TOKEN_REQUIRED=false`
- [ ] 生产灰度环境已准备 `INTERNAL_TOKEN_REQUIRED=true`
- [ ] 网关入口可正常返回 `X-Request-Id`、`X-Trace-Id`

## 发布中

- [ ] 通过网关调用 `/api/auth/login` 成功
- [ ] 开启强校验后，直连 Biz/Ops 且不带令牌请求被拒绝
- [ ] 关键日志中 `requestId/traceId` 可串联调用链

## 发布后

- [ ] 内网 ACL 已限制对 Biz/Ops 直连访问
- [ ] 令牌轮换计划已登记（建议 30~90 天）
- [ ] 巡检记录已归档（时间、环境、执行人、结论）
