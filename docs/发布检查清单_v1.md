# 发布检查清单 v1

发布前：
- [ ] `./scripts/release_precheck.sh` 通过
- [ ] `./scripts/security_baseline_check.sh` 通过
- [ ] `./scripts/observability_smoke.sh` 通过
- [ ] Nacos 配置已审核（包含 Sentinel 规则）
- [ ] 告警规则已加载（Prometheus）
- [ ] 回滚版本与镜像 tag 已确认

发布中：
- [ ] 先灰度 5%-10%
- [ ] 观察核心指标 10-15 分钟
- [ ] 再放量到 50%
- [ ] 最后全量

发布后：
- [ ] 关键链路回归（登录/入队/看板）
- [ ] 5xx、P95、429 比例符合预期
- [ ] 无 dead-letter 异常增长
- [ ] 发布记录归档

一键巡检（默认执行发布+安全+观测）：

```bash
./scripts/enterprise_readiness_check.sh
```

如果需要同时检查中间件集群：

```bash
RUN_MIDDLEWARE=true ./scripts/enterprise_readiness_check.sh
```

说明：
- 脚本会尽量执行完全部已启用检查项，并输出汇总结果。
- 只要存在失败项，脚本最终会以非 0 状态退出，便于 CI/CD 阻断发布。
